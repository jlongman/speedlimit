h1. Speedlimit

<img width="160" height="200" src="https://github.com/mschrag/speedlimit/raw/master/SpeedLimit.png" alt="sign" />

SpeedLimit is a (pf/post-10.10) preference pane for limiting your network 
bandwidth to one of a couple different speeds -- 768k DSL, Edge, 3G, 
and Dialup.  

It was adapted from the ipfw version here: https://github.com/mschrag/speedlimit

h2. Instructions

You should just be able to build the xcodeproj and install the 
resulting build product .prefPane.

h2. Download

Binaries are available in https://github.com/jlongman/speedlimit/releases/
